02 · INITIATE LINK

Get the bot online in five steps

Each step calls out the most common error you'll hit and how to clear it. Total time: under 10 minutes.

  1. 01/

    Add the bot to your server

    Open the invite link from linespolice-cad.com and select your guild. Discord will ask for the permissions View Channel, Send Messages, and Embed Links — all required.

    TIP You'll need Manage Server on your own Discord account to install bots in that guild.

  2. 02/

    Connect your Discord to Lines Police CAD

    Sign in at linespolice-cad.com, open Account → Connections, and link Discord. Without this, every command replies with You are not logged in.

  3. 03/

    Join an active community

    Dispatch, panic, search, and the economy stack all run against your active community. Join one on the website; the bot reads it on the fly.

    WATCH FOR You must join a community to use this command.

  4. 04/

    Lock down channels & roles optional

    Run /channels set channel:#dispatch to scope the bot to a specific channel, and /roles set role:@Officer to gate commands behind a Discord role. Both require the caller to have Manage Server.

  5. 05/

    Run /help and you're live

    Type / in any allowed channel — every command in the table below should now appear in Discord's autocomplete. Hit /panic to confirm the round-trip.

04 · FAQ

Common rejections, decoded.

"You are not logged in." — what now?

Your Discord account isn't linked to a Lines Police CAD account. Sign in at linespolice-cad.com, open Account → Connections, and connect Discord. Then retry the command.

"You must join a community to use this command."

Almost every command runs against your active community (dispatch status, search, economy, etc.). Join one on the website, then re-run.

"You are not allowed to use the bot in this channel."

A server admin has restricted the bot via /channels set. Run /channels view in any channel the bot can post in to see the allowlist, or have an admin add the current channel.

"You don't have permission to use this command."

A server admin gated the bot behind specific Discord roles. Run /roles view to see which roles are allowed.

"Economy is not enabled in your community."

The economy suite (/clock-in, /wallet, /inbox, /pay-fine, /contest-fine) requires the community owner to enable economy on the website.

"You already have an active shift. Use /clock-out first."

Only one shift can be active at a time. Run /clock-out to end it, then /clock-in again for the new department.
